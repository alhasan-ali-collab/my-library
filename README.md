# مكتبتي التعليمية

منصة تعليمية لعرض الكتب والوحدات الدراسية.

## المميزات

* إدارة الكتب والوحدات
* رفع صور وملفات PDF
* تصنيف حسب الفصل والصف والمادة
* عرض الكتب داخل المتصفح

## التقنيات المستخدمة

* HTML
* CSS
* JavaScript
* Node.js
* Express

## التشغيل

```bash
npm install
node server.js
```
